# javaTicTacToe

# Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

```
Example 1:

> XXXOO__O_
---------
| X X X |
| O O _ |
| _ O _ |
---------
X wins

Example 2:

> XOXOXOXXO
---------
| X O X |
| O X O |
| X X O |
---------
X wins

Example 3:

> XOOOXOXXO
---------
| X O O |
| O X O |
| X X O |
---------
O wins

Example 4:

> XOXOOXXXO
---------
| X O X |
| O O X |
| X X O |
---------
Draw

Example 5:

> XO_OOX_X_
---------
| X O   |
| O O X |
|   X   |
---------
Game not finished

Example 6:

> XO_XO_XOX
---------
| X O _ |
| X O _ |
| X O X |
---------
Impossible

Example 7:

> _O_X__X_X
---------
|   O   |
| X     |
| X   X |
---------
Impossible

Example 8:

> _OOOO_X_X
---------
|   O O |
| O O   |
| X   X |
---------
Impossible
```
